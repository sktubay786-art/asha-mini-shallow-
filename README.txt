Asha Mini Shallow V30 Final App

Final fixes included:
- Payment and settlement entries can be edited or deleted from chat/ledger.
- User can be edited or deleted from More menu.
- Bill can be deleted from history/chat.
- Reminder + Bill image share uses Web Share API where supported.
- WhatsApp Text opens exact phone number with reminder.
- QR can be dynamic or uploaded static QR.
- QR upload/change/remove added in Settings.
- Note/remarks show on bill and reminder text.
- Payee and country code are hidden/auto handled.
- Multiple bill templates added.
- Village-wise and customer-wise due report added.
- Android app with iOS-like UI style.

Important: browser apps cannot force WhatsApp to auto-send an image to an exact number. Use Reminder + Bill for image share sheet and WhatsApp Text for exact number message.
