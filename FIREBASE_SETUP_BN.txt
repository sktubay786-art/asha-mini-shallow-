Firebase free cloud sync setup (Bangla):

1) Firebase Console খুলুন
2) Create project
3) Build > Firestore Database > Create database
4) Web app add করুন (</>)
5) আপনার firebaseConfig object copy করুন
6) App open করে Settings > Cloud Sync এ paste করুন
7) Cloud Doc ID দিন, যেমন: ekramul-main
8) Test চাপুন
9) Push to Cloud চাপলে local data cloud-এ যাবে
10) Pull from Cloud চাপলে cloud data app-এ আসবে
11) Auto Cloud Sync ON করলে save হলেই কিছু সময় পরে auto push হবে

ফ্রি deploy:
A) GitHub Pages
- files upload করুন
- Pages ON করুন

অথবা
B) Firebase Hosting
- npm install -g firebase-tools
- firebase login
- firebase init hosting
- public folder হিসেবে এই app folder select করুন
- firebase deploy

যদি চান, পরের ধাপে আপনি Firebase config দিলে আমি ready-made connected version-ও বানিয়ে দিতে পারি।
